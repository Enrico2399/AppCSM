# CSM Digitale (Ionic + Angular)

## Requisiti
- **Node.js** \(consigliato LTS ≥ 18\)
- **npm**
- **Ionic CLI**

Installazione Ionic CLI (se manca):

```bash
npm install -g @ionic/cli
```

## Installazione dipendenze
Da questa cartella (`CSM-App/CSM-App`):

```bash
npm install
```

## Avvio in locale (dev server)

```bash
ionic serve
```

L’app sarà disponibile su `http://localhost:8100`.

### Porta diversa (se 8100 è occupata)

```bash
ionic serve --port 8101
```
